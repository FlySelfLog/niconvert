#########
Niconvert
#########

简介
====

niconvert 是一个弹幕下载和转换工具。把弹幕网站的评论转换成 ass 字幕格式，以便在桌面播放器使用高画质视频看吐槽。

.. image:: http://static.qixinglu.com/github/images/niconvert/gtk.png

支持如下特性：

* 支持 acfun 和 bilibili 。

* 解析下载无需注册和积分。

* 自定义字幕样式。

* 多种界面，包括 cli/tk/gtk/qt/wxwidget，支持 Linux/Mac/Windows。
